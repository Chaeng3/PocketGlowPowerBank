# IP5353（21700电池）开源快充充电宝

> Type-C 单进单出 / 22.5W 快充 / 21700 电池 / 18650 电池夹结构 / 高亮长续航照明

本项目是一款基于 **IP5353** 的开源快充充电宝，采用单节 **21700 锂电池**作为储能单元，支持 **Type-C 单进单出**和最高 **22.5W 快充**。

项目使用 **18650 电池夹**作为 21700 电池两端的固定与导电结构，以节省内部空间，并加入了高亮照明功能。低功耗照明模式下，可实现约 **10 天续航**。

## 硬件说明

### 1. IP5353 电源管理部分

本项目使用 **IP5353** 作为核心电源管理芯片，主要负责单节锂电池充电、升压输出、快充协议识别、电量显示和按键控制等功能。

当 Type-C 接口接入充电器时，IP5353 对内部 21700 锂电池进行充电管理；当 Type-C 接口接入外部设备时，IP5353 将电池电压升压后对外输出。

IP5353 的集成度较高，外围电路相对简洁，适合制作小体积快充充电宝。

---

### 2. Type-C 单进单出设计

本项目只保留一个 **USB Type-C** 接口，既作为输入接口，也作为输出接口。

相比常见的多接口充电宝方案，单 Type-C 设计具有以下优点：

- 接口数量更少
- 外壳开孔更简单
- PCB 布局更紧凑
- 整体外观更简洁
- 使用方式更符合现在的主流设备习惯

---

### 3. 21700 电池 + 18650 电池夹结构

本项目实际使用 **21700 锂电池**作为储能单元。相比常见 18650 电池，21700 电池通常具有更高容量，更适合用于移动电源和长续航照明场景。

为了节省空间，项目没有使用传统 21700 电池座，而是使用 **18650 电池夹**作为电池两端的固定与导电结构。

这种结构的优点：

- 节省内部空间
- 降低整体厚度
- 方便安装和拆卸电池
- 方便更换电芯
- 更适合紧凑型外壳设计

使用时需要注意电池夹位置、电池长度、电池接触可靠性和外壳限位结构，避免电池松动、虚接或接触不良。

---

### 4. 电量指示与按键控制

项目设计有 LED 电量指示功能，可用于显示当前电池电量或工作状态。

轻触按键可用于：

- 唤醒输出
- 查看电量
- 控制照明功能
- 切换相关工作状态

---

### 5. 高亮 LED 照明

本项目加入了高亮 LED 照明功能，可用于：

- 夜间临时照明
- 停电应急照明
- 户外露营
- 桌面辅助照明
- 备用应急灯

在低功耗照明模式下，配合 21700 电池容量，可实现约 **10 天不充电续航**。

## 注意事项

- 本项目涉及锂电池充放电，使用时请注意电池安全
- 请使用质量可靠的 21700 锂电池
- 电池安装前必须确认正负极方向，避免反接
- 本项目使用 18650 电池夹固定 21700 电池，需要特别注意电池夹位置和接触可靠性
- **IP5353 不建议在立创商城购买，立创商城销售的版本为带 I2C 的型号，与本项目使用的非 I2C 版本不一致，直接替换可能导致功能异常**
- 焊接 IP5353 前请仔细核对芯片丝印、封装和具体型号，确认与原理图版本一致
- 个性贴纸可在淘宝购买，但图案较多，长时间挑选容易眼睛看花，建议提前确定尺寸和风格
- 首次上电前，请检查 BAT、VBUS、GND 等关键网络是否短路
- Type-C 接口焊接完成后，应检查是否存在虚焊、连锡或引脚短路
- IP5353、电感、电容等电源器件需要正确焊接，尤其是功率电感不能虚焊
- 大电流路径应尽量保证走线宽度，避免发热和压降过大
- 高亮 LED 工作时可能产生一定热量，长时间使用应注意散热
- 照明续航时间会受到电池容量、LED 功率和使用模式影响
- 快充输出时芯片、电感和 PCB 局部发热属于正常现象，但不应出现异常高温
- 如果用于长时间大功率输出，建议进行温升测试
- 本项目为开源学习项目，实际使用前请自行完成安全性和可靠性验证
